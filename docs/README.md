# Kiln documentation

Welcome to the Kiln documentation

> [!NOTE]
> This documentatoin set is work in progress. Feel free to contribute to 

Kiln is a modern **build system** that consumes `CMakeLists.txt`, interpret the CMake language, collects the build graph, managing dependencies and executing the build process. Kiln is intentionally designed to act close to how `cmake` works. But with a focus on eliminating common footguns.
